## Getting Started

### 1. Clone this Repository

```
git clone https://github.com/bobarros/11typroject.git

```

### 2. Navigate to the directory created

```
cd 11typroject
```

### 3. Install dependencies

```
npm install
```

### 4. Run Eleventy

```
npm run dev
```

### 5. Build for production

```
npm run build
```
